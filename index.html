Estructura if, forEach y setTimeout

  • Estructura if
    • Estructura basica 
      •                         
                          const num1 = 5;
                          const num2 = 10;
                          if( num1 > num2 )  // condicion  
                          {
                              console.log( num1 + ' es mayor que ' + num2 );  // true  
                          } else {
                              console.log( num2 + ' es mayor que ' + num1  );  // false  
                          }
    • Operador ternario 
      •                         
                          const num1 = 5;
                          const num2 = 10;
                          num1 > num2 ? console.log( num2 + ' es mayor que ' + num1  ) : console.log( num2 + ' es mayor que ' + num1  );
                          // condicion  ?  true  :  false;
  • Estructura forEach 
    •                     
                      array.forEach(element => {
                            console.log(element);
                      });
  • Estructura setTimeout 
    •                     
                      setTimeout( () => {
                            console.log('Hola! Tarde 1s en aparecer.');
                      }, 1000);

Arreglos

  • Crear Arreglos 
    •                     
                      const frutas = [ 'Manzana' , 'Banana' ];
  • Recorrer Arreglos 
    •                     
                      const frutas = [ 'Manzana' , 'Banana' ];
                      frutas.forEach(fruta => {
                            console.log(fruta);
                      });
  • Metodos
    • Conocer el largo 
      •                         
                          frutas.length;
    • Acceder a un elemento del arreglo mediante su indice 
      •                         
                          frutas[0];
    • Añadir un elemento 
      •                         
                          frutas.push('Fresa');
    • Eliminar el último elemento 
      •                         
                          frutas.pop();
    • Añadir un elemento al principio 
      •                         
                          frutas.unshift('Fresa');
    • Eliminar el primer elemento 
      •                         
                          frutas.shift();
    • Encontrar el indice de un elemento 
      •                         
                          frutas.indexOf('Banana');
    • Eliminar un elemento mediante su posicion 
      •                         
                          frutas.splice(pos, 1);
    • Eliminar varios elementos a partir de una posicion 
      •                         
                          const vegetales = [ 'Repollo' , 'Nabo', 'Rábano', 'Zanahoria' ];
                          const pos = 1;
                          const numElementos = 2;
                          vegetales.splice(pos, numElementos);
    • Crear una copia 
      •                         
                          let copiaArray = frutas.slice();

Buscar elementos del DOM

  • Buscar con getElement
    • Buscar por Id 
      •                         
                          const element = document.getElementById('miId');
    • Buscar por Name 
      •                         
                          const element = document.getElementsByName('miName');
    • Buscar por TagName 
      •                         
                          const element = document.getElementsByTagName('miTagName');
    • Buscar por ClassName 
      •                         
                          const element = document.getElementsByClassName('miClassName');
    • Buscar por TagNameNS 
      •                         
                          const element = document.getElementsByTagNameNS('miTagNameNS');
  • Buscar con querySelector o querySelectorAll
    • Buscar por Id 
      •                         
                          const element = document.querySelector('#miId');
    • Buscar por Name 
      •                         
                          const element = document.querySelector('[name="miName"]');
                          const elements = document.querySelectorAll('[name="miName"]');
    • Buscar por TagName 
      •                         
                          const element = document.querySelector('miTagName');
                          const elements = document.querySelectorAll('miTagName');
    • Buscar por ClassName 
      •                         
                          const element = document.querySelector('.miClassName');
                          const elements = document.querySelectorAll('.miClassName');
    • Buscar por Data 
      •                         
                          const element = document.querySelector('[data="miData"]');
                          const elements = document.querySelectorAll('[data="miData"]');
    • Buscar por Alt 
      •                         
                          const element = document.querySelector('[alt="miAlt"]');
                          const elements = document.querySelectorAll('[alt="miAlt"]');
  • Busqueda más específica con querySelector o querySelectorAll
    • Buscar un elemento que contenga x clase 
      •                         
                          const element = document.querySelector('input', '.miClase');
                          const elements = document.querySelectorAll('input', '.miClase');

Navegar por el DOM

  • Navegar a través de elementos
    • Devuelve una lista de elementos HTML hijos 
      •                         
                          element.children;
    • Devuelve el padre del elemento o null si no tiene 
      •                         
                          element.parentElement;
    • Devuelve el primer elemento hijo 
      •                         
                          element.firstElementChild;
    • Devuelve el último elemento hijo 
      •                         
                          element.lastElementChild;
    • Devuelve el elemento hermano anterior o null si no tiene 
      •                         
                          element.previousElementSibling;
    • Devuelve el elemento hermano siguiente o null si no tiene 
      •                         
                          element.nextElementSibling;
  • Navegar a través de nodos
    • Devuelve una lista de nodos hijos. Incluye nodos de texto y comentarios 
      •                         
                          element.childNodes;
    • Devuelve el nodo padre del nodo o null si no tiene 
      •                         
                          element.parentNode;
    • Devuelve el primer nodo hijo 
      •                         
                          element.firstChild;
    • Devuelve el último nodo hijo 
      •                         
                          element.lastChild;
    • Devuelve el nodo hermano anterior o null si no tiene 
      •                         
                          element.previousSibling;
    • Devuelve el nodo hermano siguiente o null si no tiene 
      •                         
                          element.nextSibling;

Crear, modificar o eliminar elementos HTML

  • Crear elementos html 
    •                     
                      const nuevoDiv = document.createElement('div');
  • Eliminar elemento html 
    •                     
                      nuevoDiv.remove();
  • Agregar elemento hijo 
    •                     
                      const nuevoTexto = document.createTextNode('Hola! ¿Qué tal?');
                      nuevoDiv.appendChild(nuevoTexto);
  • Eliminar elemento hijo 
    •                       
                        nuevoDiv.removeChild(element);
  • Cambiarle los estilos a un elemento 
    •                     
                      nuevoDiv.style.backgroundColor = 'red';
  • Agregarle una clase a un elemento 
    •                     
                      nuevoDiv.className = 'nombreClase';
  • Agregarle una classList a un elemento 
    •                     
                      nuevoDiv.classList.add('nombreClase');
                      nuevoDiv.classList.add('nombreClase1', 'nombreClase2');
  • Eliminar una classList de un elemento 
    •                     
                      nuevoDiv.classList.remove('nombreClase');
                      nuevoDiv.classList.remove('nombreClase1', 'nombreClase2');
  • Ver si el elemento contiene x clase en su classList 
    •                     
                      nuevoDiv.classList.contains('nombreClase');
  • Insertar elementos html con innerHTML
    • Devuelve el nombre del nodo (etiqueta si es un elemento HTML). Sólo lectura 
      •                         
                          nuevoDiv.nodeName;
    • Devuelve el contenido de texto del elemento. Se puede asignar para modificar 
      •                         
                          nuevoDiv.textContent;
    • Devuelve el contenido HTML del elemento. Se puede usar asignar para modificar 
      •                         
                          nuevoDiv.innerHTML;
    • Idem a .innerHTML pero incluyendo el HTML del propio elemento HTML 
      •                         
                          nuevoDiv.outerHTML;
    • Versión no estándar de .textContent de Internet Explorer con diferencias. Evitar 
      •                         
                          nuevoDiv.innerText;
    • Versión no estándar de .textContent/.outerHTML de Internet Explorer. Evitar 
      •                         
                          nuevoDiv.outerText;
